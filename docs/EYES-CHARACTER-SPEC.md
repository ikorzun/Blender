# Персонаж-миксер: эмоции и конструкция (спека 2026-07-21)

Направление ИНТЕРФЕЙС. Решения владельца от 2026-07-21: конструкция стоит
**своей строкой под чипами**, **время партии убрано**, **зрачки отдельным
слоем**. Умолчания, взятые без возражений: полоска сохраняет обе роли
(накопление серии → остаток турбо), цифры системным шрифтом
(`font-variant-numeric: tabular-nums`, 0 КБ веса).

## 1. Матрица эмоций

Семь состояний. Приоритет сверху вниз — первое совпавшее выигрывает.

| # | Событие в игре | Эмоция | Держится | Источник в коде |
|---|----------------|--------|----------|-----------------|
| 1 | победа (все предметы спасены) | **добрые** | до закрытия экрана | `level.over` + пусто |
| 2 | поражение | **закрытые** | до закрытия | `level.over` |
| 3 | турбо (power chain) | **удивлённые** | весь режим | `chainUntil > now` |
| 4 | раскопан сюрприз | **удивлённые** | 1 с | ⚠️ ждёт хук, см. §5 |
| 5 | миксер мелет | **злые** | пока крутятся лопасти | `grinding` |
| 6 | ≤3 с до перемолки | **хитрые** | до перемолки | `idleLimit − idle ≤ 3` |
| 7 | горит серия/комбо | **добрые** | пока горит | `comboUntil > now` |
| 8 | простой > 8 с | **закатанные** | до действия | `idle > 8` |
| 9 | тап по глазам | **хитрые** + подскок | 0.8 с | `faceEvent('sly', 800)` |
| 10 | старт уровня / интро | **спокойные** | — | `intro` |
| 11 | обычная игра | **спокойные** | — | иначе |
| — | моргание (фон) | **закрытые** | 120 мс раз в 4–7 с | только в спокойных/добрых |

Лесенка нарастания угрозы читается без слов:
**спокойные → закатанные → хитрые → злые**. Это и есть телеграф перемолки
характером, вместо красной плашки (её владелец убрал 2026-07-19).

В Easy (терпение 30 с) успевают все три ступени; в Hard (10 с) «закатанные»
пропускаются — простой сразу переходит в «хитрые» на 7-й секунде. Так и
задумано: в Hard скучать некогда.

## 2. Геометрия конструкции

**Почему своя строка.** Замер свободной ширины между левой и правой
группами чипов (headless, боевая сборка):

| ширина | ур.1 | ур.3 (типичный) | ур.12 |
|--------|------|-----------------|-------|
| 320 px | 6 px | **0 px** | 0 px |
| 360 px | 46 px | **2 px** | 0 px |
| 390 px | 76 px | 32 px | 0 px |

По центру верхнего бара конструкции жить негде — уже на 3-м уровне глаза
съедали таймер и монеты.

**Текущий бокс (после переноса, замер на боевой сборке):**

| параметр | значение |
|----------|----------|
| ширина | 220 px (фикс) |
| положение | `top: safe-area + 56px`, центр по горизонтали |
| высота занятая | 123 px (y 56…179) |
| зазор до чипов сверху | 7 px |
| поля по краям на 320 px | 50 px с каждой стороны |
| порядок сверху вниз | глаза → отсчёт до перемолки → полоска турбо |
| глаза | 71 px |
| полоска | 140 × 6 px, радиус 3 |

Ниже 179 px начинается пустая верхняя часть чаши — туда конструкция
заходить может, предметы там не лежат (верх кучи ≈ y 230 на 780).

## 3. В каком виде отдавать ассеты

**Формат: SVG, 7 файлов** — по одному на эмоцию.

- **Одна канва и один `viewBox` на все семь**, глаза в одинаковых
  координатах. Иначе при смене эмоции картинка прыгает.
- **Зрачки — отдельными группами** с id `pupil-l` и `pupil-r` (решение
  владельца). Тогда взгляд сможет следить за тапом почти бесплатно:
  трансформ по группе, без перерисовки.
- Прозрачный фон, без растров и внешних шрифтов внутри файла.
- Прогнать через SVGO. **≤8 КБ файл, ≤60 КБ на все семь** — весь проект
  инлайнится в один `index.html`, бюджет ~1 МБ.
- Имена: `eyes-calm`, `eyes-kind`, `eyes-angry`, `eyes-surprised`,
  `eyes-closed`, `eyes-sly`, `eyes-rolled`.
- ⚠️ **Экспорт из панели Export, не из Dev Mode.** Dev Mode отдаёт
  `preserveAspectRatio="none"` и `width/height="100%"` — такие файлы
  рендерятся растянутыми.

Если стиль растровый — PNG @3x с прозрачностью, ≤40 КБ каждый (base64
добавит ещё треть веса).

**Подложка/рамка конструкции** (если она есть в дизайне) — отдельным SVG.
Полоску турбо и цифры не рисовать: заполнение анимируется, цвет плывёт из
JS, это CSS.

## 4. Что уже сделано (v1-test-13)

Скелет собран и работает на эмодзи-заглушках — ассеты вставляются без
переделки логики:

- `#face` в `shell.html` — одна конструкция (глаза + `#mixerTimer` +
  `#chainBar`), своей строкой под чипами.
- `#mixerTimer` переехал из нижнего бара (отменяет спеку 2026-07-20-б).
- Чип времени партии `#timer` удалён вместе с его обновлением в `99-main`.
- `85-hud.js`: `eyesMood` отдаёт ИМЯ состояния из `FACE_STATES`, отрисовка
  вынесена в `setFace` — при приезде SVG меняется только эта функция и
  таблица `FACE_GLYPH`. Плюс `tickFace` (каждый кадр: моргание + полоска)
  и `faceEvent(state, ms)` для коротких реакций.

Проверено headless: моргание ловится в покое; серия → добрые; ≤3 с до
перемолки → хитрые; помол → злые; настоящая цепная реакция → удивлённые;
тап → хитрые и возврат через 0.8 с. Геометрия на 320/360/390 без
столкновений.
